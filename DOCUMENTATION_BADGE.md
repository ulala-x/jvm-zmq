# Documentation Badge for README

Add this badge to your main README.md to provide a quick link to the API documentation.

## Badge Markdown

```markdown
[![API Documentation](https://img.shields.io/badge/API-Documentation-blue)](https://ulala-x.github.io/jvm-zmq/)
```

Add a section like this to your README.md:


## After First Deployment

After the first successful deployment to GitHub Pages:

1. Verify the documentation is accessible at: https://ulala-x.github.io/jvm-zmq/
2. Update the badge URLs if needed
3. Add the badge to the top of your README.md
4. Test all documentation links
